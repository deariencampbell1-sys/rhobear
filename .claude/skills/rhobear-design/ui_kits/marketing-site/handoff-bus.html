SECTION · SUBSTRATE · 12 / 20

Crew hands work off. You don't shuttle messages.

The handoff bus is the message floor where agents pass work to each other — envelopes, results, leave-behinds, and aggregated lane outcomes — without going through you.

HANDOFF ROUTER

Agents can pass work to each other through RHOBEAR instead of waiting for you to copy messages around.

TECH

A typed handoff layer with envelope routing, ack/nack semantics, retry budgets, and observability for inter-agent calls.

RESULT ENVELOPE

Every worker reports back in a format RHOBEAR can understand, track, and act on.

TECH

Structured result schema for status, evidence, diffs, side-effects, risks, and follow-up actions surfaced uniformly to the judge.

DURABLE MESSAGE BUS

Agents can leave messages, tasks, and results for each other without losing context.

TECH

Persistent, ordered, replayable agent-to-agent message bus that survives crashes, restarts, and runtime swaps.

LANE-SET AGGREGATOR

When several agents work in parallel, RHOBEAR rolls their progress into one clear outcome.

TECH

Lane-set rollup that merges concurrent worker reports into one aggregate envelope the judge and owner can read in a single glance.
